# Non-Homebrew Dependencies

This file is for setting up non-Homebrew dependencies.

To use this file, follow the instructions in each installation section.

## Rust

Install `rustup` (which also automatically installs `rustc` and `cargo`) with:

```bash
curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
```

Official installation instructions can be found at [The Rust Programming Language](https://doc.rust-lang.org/book/ch01-01-installation.html).

## uv

Install `uv` with:

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

Official installation instructions can be found at [uv](https://docs.astral.sh/uv/getting-started/installation/).
